  • Hardware Computing Group, Microsoft Research Asia, 2014.11-2015.4
    Project : Wearable PM2.5 Sensing Arm Band Using Machine Learning
    Prediction and calibration of PM2.5 concentration using fully-connected neural networks. This work estimates the concentration of PM2.5 with NN using the concentration of related air pollutions that are easier to be sensed. I proposed a distribution normalization method to preprocess the input data and emphasize the important highly-polluted data. The proposed sensor obtained a more than 0.95 correlation with ideal value.
    This work got the “Stars of Tomorrow” award of MSRA internship program.

  • Smartbow Information Technology Co., Ltd., 2012.6-2013.7
    Project: Structural Health Monitoring System of Ronghu Bridge
    Lixue Xia mainly focused on the high-availability gateway design for wireless sensor network (WSN) used in big data applications. To tolerant the connection failure between WSN and remote server, a high-availability gateway that can store and process temporary data was designed. This work has been practically integrated in to the Structural Health Monitoring System of Ronghu Bridge (v3.0) and other projects.
    This work got the outstanding B.S. thesis award of E.E. Dept., Tsinghua University.
